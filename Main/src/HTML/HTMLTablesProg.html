HTML Tables

An HTML table is defined with the 'table' tag. Each table row is defined with the tr tag. A table header is defined with the th tag. By default, table headings are bold and centered. A table cell is defined with td tag. e.g. one row for header, one row for columns, one row for footer, etc

Output

Jill Smith 50
Eve Jackson 94
